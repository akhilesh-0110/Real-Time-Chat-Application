# Real-Time Chat & Calling Application

A full-stack **Real-Time Chat & Calling Application** that enables users to exchange instant messages and make real-time audio/video calls. 
Built with modern web technologies, the project demonstrates expertise in real-time communication, secure authentication, WebSocket architecture, 
and responsive frontend development.

---

## Key Features

### Authentication

* Secure User Registration & Login
* JWT Authentication
* Protected Routes
* Password Encryption using bcrypt

### Real-Time Chat

* Instant One-to-One Messaging
* Live Message Delivery
* Online & Offline Presence
* Typing Indicators
* Message Timestamps
* Conversation History
* User Search

### Audio & Video Calling

* Real-Time Audio Calling
* Real-Time Video Calling
* Incoming Call Notifications
* Accept / Reject Calls
* End Call Functionality
* Camera & Microphone Controls
* Call Status Updates
* Peer-to-Peer Communication for Low Latency

### User Experience

* Responsive UI for Mobile & Desktop
* Modern Chat Interface
* Loading States
* Error Handling
* Smooth Navigation

Tech Stack

### Frontend

* React.js
* Tailwind CSS
* React Router
* Axios

### Backend

* Node.js
* Express.js
* Socket.io
* JWT Authentication
* bcrypt.js

### Database

* MongoDB
* Mongoose

### Real-Time Communication

* Socket.io
* WebRTC (for Audio/Video Calling)

---

## Highlights

* Designed a scalable real-time communication architecture using Socket.io.
* Implemented low-latency audio and video calling with WebRTC.
* Built secure authentication using JWT and encrypted passwords.
* Optimized real-time messaging and call signaling for smooth user experience.
* Developed responsive UI compatible across desktop and mobile devices.
* Structured the project using modular architecture for maintainability and scalability.
